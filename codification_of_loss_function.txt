We describe below the codifications of the different cost functions as well as the value of their hyperparameter useful to reimplement these codes. Please refere to the original article for more details.

Codification for Hard Mining Loss:

'hmld10000g05': hard mining loss with delta parameter equal to 10000, gamma parameter equal to 0.5, and alpha parameter equal to 1
'hmld100000g2a75: hard mining loss with delta parameter equal to 100000, gamma parameter equal to 2, and alpha parameter equal to 0.75

Codification of SOA loss functions:

'ce' : cross entropy
'csce' : cross entropy weighted by inverse class frequencies
'flg1': focal loss with gamma parameter equal to 1
'flg2a50': alpha focal loss with gamma parameter equal to 2 and alpha equal to 0.5
'CBLoss': cross entropy loss weighted by the effective number of sample
'CBFLossg05': focal loss weighted by the effective number of sample with gamma equal to 0.5
'LDAM-RW': implementation of LDAM loss
